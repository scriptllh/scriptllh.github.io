记mac 迁移操作

  1. time machine 失败
    • 开机按 command + R 进入
    • 来到OS X 实用工具，选择“从 Time Machine 备份进行恢复”，点击“继续”
    • 成功检测到Time Machine备份后，选择想要恢复的版本
    • 然后进行恢复（可惜的是每次恢复到一半就失败了，暂时不知道什么原因，后来选择了迁移助理）
  2. 迁移助
    • 开机按 command + R 进入
    • 来到OS X 实用工具，选择重新安装OS X
    • 进入该页面
    • 打开老mac 上打开迁移助理，选择 to another mac
    • 在新mac上选择，如图
  3. 然后开始迁移

  • 迁移成功后会重启电脑

  • 注意:重启后输入电脑密码之后，需要等半天（不要以为坏了，只是在加载系统和配置），最后就ok了
